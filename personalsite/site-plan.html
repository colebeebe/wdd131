Home page

Hello, I'm Cole Beebe-Urbanawiz. Originally from Idaho, I'm now living with my wife and baby in the Atlanta area. I enjoy expressing my creativity through passion projects. When I'm not working on these projects, I'm usually spending time with my daughter or playing video games.

Projects

Voxel-Style Rendering Engine

Written in C# and utilizing the OpenGL mappings provided by OpenTK, this is an engine that has been in development for nearly half a year. As of yet, it is capable of rendering millions of blocks at a sustainable 1500+fps. The purpose of this project is to explore 3D rendering pipelines.

Wii Portable

A complete trim and relocation of many components of a Nintendo Wii console. This project included lots of micro-soldering and extreme precision. The console can run for about three hours on battery, and played anywhere.

Atari Asteroids/Moonlander Arcade

This project was created using the Godot game engine. It recreates the arcade versions of Asteroids and Moonlander. The purpose of this project was to learn more about game development.
